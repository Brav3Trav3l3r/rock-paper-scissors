
# Rock-Paper-Scissors game

A rock-paper-scissors game, where you play against comp. Made with vaniall Js and css

## Run Locally

Clone the project

```bash
  git clone https://github.com/Brav3Trav3l3r/rock-paper-scissors
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
